# BladeBurn

Маленькое Laravel приложение для создания зашифрованных сообщений и для их чтения. При прочтении сообщение удаляется

## Стек
- PostgreSQL
- Redis
- Docker (Laravel Sail)
- Tailwind CSS

## Фичи
- Сообщения шифруются в бд, обеспечивая надежность и безопасность
- UUID в качестве первичных ключей
- Rate Limiting (защита от спама в сторону api)
- Background jobs (уведомления на почту о прочтении сообщений через очереди)
- Scheduler (автоочистка просроченных сообщений)

## Запуск

docker compose up -d + migrate
